【原创】Introducing 猫猫短链接

发表于 2017-01-25 | 分类于 sharing |

此项目仍然处于beta状态，不保证数据安全！！请自行备份数据！！

短链接这个项目我可以说是策划了很久很久了，市面上的短链接有几个毛病，先不管它用的人多不多，很多短链接网站参杂大写小写字母，因为短链接最早应该是为了发Twitter而用的所以要把所有能用的字符都用上，所以还算有道理。

但是我有时候用短链接的需求不一样，是为了把长链接写在纸上，这样的话大小写字母很容易分不清，包括大写i和小写L数字1之类的，另外有时候想分享给别人的是一段话（甚至是一段带格式的话），而不一定是一个链接（有点像Pastebin，但是如果能带格式那就更好了）。目前还没有一个链接缩短器可以做成这样。

这个月我把这坑给填了，下定决心把这东西写出来，终于它就成了我的这个网站：https://lk1.bid

什么是短链接？

Wikipedia: （网址缩短服务）可以提供一个非常短小的URL以代替原来的可能较长的URL，将长的URL地址缩短。用户访问缩短后的URL时，通常将会重定向到原来的URL。

短网址最初是由于微博服务的字数限制而诞生的，用户访问缩短的链接后会被重定向（跳转）到原链接，也就是说访问短网址等于访问原链接。短网址可以便于记忆或个性化。

常见的短链接服务有 t.co、bit.ly、goo.gl

什么是Pastebin？

Wikipedia: A pastebin is a type of web application where users can store plain text.

Pastebin 是一个用户用来存储文本的网络应用。一般Pastebin用于储存程序代码用于分享，但是可以存储任何文本。

常见的Pastebin服务有 http://pastebin.com/

猫猫短链接的功能

猫猫短链不仅是一个链接缩短器，还是一个Pastebin，并且Pastebin里不仅可以存放纯文本，还可以有Markdown格式化的文本。

  • 生成短链接
  • 粘贴文本
  • Markdown格式化文本
  • Telegram Bot支持

使用方法

将长链接粘贴到首页的文本框中，点击提交便可以复制缩短的链接。短链接将会直接跳转到原网站。

将文本粘贴到首页的文本框中，点击提交便可以复制缩短的链接。访问短链接将会显示提交的内容

链接格式：lk1.bid/[ID]/[格式] 其中ID由数字和小写字母组成，格式为 plain/md/raw 中其一，分别代表文本、Markdown格式化文本、raw文本

未来可能会添加的功能：

  • 账号注册、短链接内容编辑管理
  • 自定义链接
  • 密码保护
